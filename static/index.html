Help

Fumula helps you track your incense ingredients, build recipes from them, and print measurement sheets. Your data is stored in your browser — nothing is saved to a server. Use Export JSON to save a local backup and Import JSON to restore it.

Ingredients

Add ingredients in the Ingredients tab. Each ingredient belongs to a category (wood, resin, binder, or any category you define). Categories are used to organize the blending stages of a recipe.

Recipes

In the Recipes tab, create a recipe by setting what percentage of the batch each category makes up, then within each category which ingredients to use and in what ratio. One ingredient or category per group can be set to auto-fill (the ⟳ button) — it adjusts automatically to keep the total at 100%.

Related recipes (e.g. two versions of the same base) can be organized into a Group using the Group field in the editor. Use New Variant to clone the selected recipe into the same group.

Printing

Click Print to generate a recipe sheet. Enter one or more target batch sizes in grams (comma-separated) and the sheet will show gram amounts for every ingredient at each batch size.

Data

Export JSON saves all your ingredients and recipes to a file. Import JSON loads them back. On a shared or deployed instance, each user's data is independent and lives in their own browser.
